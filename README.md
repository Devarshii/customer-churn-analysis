# customer-churn-analysis
This project analyzes customer churn using the IBM Telco dataset to identify key factors influencing customer retention. Through data analysis and statistical testing, it highlights patterns such as higher churn among month-to-month users, new customers, and high-paying segments, along with actionable business recommendations to reduce churn.


# Business Problem

Customer churn is a major challenge for subscription-based businesses. The goal of this project is to identify the key factors that lead to customer churn and provide actionable insights to improve customer retention.

# Dataset

* IBM Telco Customer Churn Dataset
* ~7,000 customer records
* Includes customer demographics, subscription details, and billing information

# Approach

1. Data Cleaning

   * Converted data types
   * Handled missing values

2. Exploratory Data Analysis (EDA)

   * Analyzed churn distribution
   * Compared churn across different customer segments

3. Hypothesis Testing

   * t-test (Monthly Charges vs Churn)
   * Chi-square test (Contract Type vs Churn)

# Key Insights

* Month-to-month contract customers have the highest churn
* Higher monthly charges are associated with increased churn
* Customers with low tenure are more likely to leave
* Electronic check users show higher churn behavior

# Business Recommendations

* Promote long-term contracts with incentives
* Offer retention strategies for new customers
* Improve service/value for high-paying customers
* Target high-risk segments with personalized campaigns

# Tools & Technologies

* Python (Pandas, NumPy)
* Data Visualization (Matplotlib, Seaborn)
* Statistical Testing (Scipy)

# Project Outcome

This project demonstrates how data analysis and statistical methods can be used to identify churn drivers and support data-driven business decisions.
